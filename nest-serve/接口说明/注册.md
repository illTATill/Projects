# 用户信息接口文档

## 📌 注册接口：`registeruser`

用于新用户注册账号。

### 请求方式：

`POST`

### 接口地址：

`/registeruser`

### 请求参数：

| 参数名          | 类型   | 是否必填 | 说明                         |
| --------------- | ------ | -------- | ---------------------------- |
| phoneNumber     | string | 是       | 用户手机号                   |
| password        | string | 是       | 用户密码                     |
| confirmPassword | string | 是       | 确认密码，需与 password 一致 |

### 请求示例：

```json
{
  "phoneNumber": "1234",
  "password": "1erft",
  "confirmPassword": "rfthygb"
}
```
