# 用户信息接口文档

## 📌 登录接口：`loginuser`

用于用户登录账号。

### 请求方式：

`POST`

### 接口地址：

`/loginuser`

### 请求参数：

| 参数名      | 类型   | 是否必填 | 说明       |
| ----------- | ------ | -------- | ---------- |
| phoneNumber | string | 是       | 用户手机号 |
| password    | string | 是       | 用户密码   |

### 请求示例：

```json
{
  "phoneNumber": "1234",
  "password": "1erft"
}
```
